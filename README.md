## 🚧  Setup

Used libraries and how to install

yarn

```yarn
yarn add @testing-library/react @testing-library/jest-dom @testing-library/user-event -D
```

Check `jest.config.ts` for setup configuration

## 🏃‍ Running test

After `yarn` to install all dependencies
```bash
yarn test
#or
yarn test --watchAll

```
